# 🚀 Monetizator v2: Главная панель управления

Добро пожаловать в технический хаб проекта **Monetizator**. Этот сайт — высококонверсионная воронка для эксперта Сергея Осипука, построенная на принципах "Money Matrix Protocol".

## 📂 Структура документации
*   [План разработки Админки](ADMIN_PLAN.md) — ТЕКУЩИЙ ФОКУС.
*   [План развития и запусков](ROADMAP.md) — общая стратегия.
*   [Инструкция по обслуживанию](MAINTENANCE.md) — как менять контент, цены и кейсы.
*   [Технический фундамент (Antigravity)](./Документация%20по%20сайту/Техническая%20документация%20Сергей%20Осипук%20сайт.md) — детальное описание архитектуры, БД и ИИ.

## 🛠 Стек технологий
*   **Frontend**: React + Vite + Tailwind CSS
*   **Animations**: Framer Motion / Lucide Icons
*   **Infrastructure**: Supabase (DB, Auth, Edge Functions)
*   **Deployment**: GitHub Pages (через Git Subtree)

## 🚀 Быстрый старт (для разработчика)
1.  `npm install` — установка зависимостей.
2.  `npm run dev` — запуск локально.
3.  `npm run build` — сборка проекта.

## 📡 Деплой
Деплой выполняется ОДНОЙ командой из корня основного репозитория:
```bash
git subtree push --prefix sites/monetizator-v2 monetizator main
```
*Внимание: Никогда не деплойте папку напрямую, используйте только команду выше.*
