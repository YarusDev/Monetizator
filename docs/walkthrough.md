# Результаты унификации и нормализации данных

Мы провели масштабную работу по приведению сайта к единому визуальному стандарту (Matrix Protocol) и очистке структуры данных.

## 📊 База данных и Синхронизация
- **SQL Миграция**: Все данные из устаревших полей (`manifesto_text`, `footer_text`) перенесены в стандартные поля `subtitle` и `footer`.
- **Нормализация**: Теперь все блоки используют единый набор ключей JSONB (`title`, `subtitle`, `items`, `footer`), что исключает путаницу в админке.

## 🎨 Дизайн (Expert Style)
- **Manifesto & AntiTarget**: Полностью переработаны. Красные акценты заменены на изумрудные. Карточки тезисов теперь имеют премиальный вид с тенями, ховер-эффектами и улучшенной типографикой.
- **Method**: Обновлен стиль этапов, исправлены баги отображения подзаголовка.
- **Hero**: Унифицирован футер и метрики.

## 🛠 Админка (UX/UI)
- **Services**: 
  - Кнопка **"Новый продукт"** теперь ярко выделена изумрудным градиентом.
  - Иконки редактирования заменены на понятные (Pencil/Edit3) вместо старых стрелок.
  - Улучшены стили кнопок реордеринга (Up/Down).
- **Icon Editing**: Теперь во всех блоках с `StandardItem` можно менять иконки через удобный попап.
- **Quiz Visuals**: Добавлена поддержка превью-изображений в квизе. Создана уникальная "Карта ресурсов" в стиле Matrix Protocol.
- **Expert Subtitle**: В блок "Об эксперте" добавлено поле подзаголовка для более точного позиционирования (напр. "Лидер среды").
- **Stability**: Исправлены критические ReferenceError при работе с блоками Gift и Footer.

## 📇 Глобальная система контактов
- **Централизация**: Создан единый банк контактов `m_contacts`. Теперь социальные сети и мессенджеры настраиваются один раз и используются во всех блоках сайта.
- **Умные ссылки**: Реализовано авто-форматирование `tel:`, `mailto:` и прямых ссылок на Telegram/WhatsApp.
- **Управление блоками**: Блоки `Header` и `Contacts` теперь позволяют выбирать контакты из глобального списка, настраивать их видимость и порядок индивидуально для каждого блока.
- **ContactManager UX**:
  - Кнопки **Редактировать** и **Удалить** сделаны статичными и размещены рядом с переключателем активности для удобства доступа.
  - **Умная сортировка**: Контакты автоматически группируются по приоритету: сначала Активные, затем Незаполненные (требующие внимания), и в конце Скрытые.
  - Добавлен визуальный индикатор заполненности данных (серые карточки для пустых контактов).
  - Улучшена цветовая индикация статусов (Активен/Скрыт/Не заполнен).
