# 🏗 CMS Structure: Управление контентом

Логика связи между админкой и лендингом.

## 1. Матрица сущностей
- **Продукты (`m_products`)**: Содержат ID, Название, Цену, Описание, Тип.
- **Кейсы (`m_cases`)**: Содержат ID, Заголовок, Цифры результата, Суть.

## 2. Связывание (Linking)
- В настройках секций (например, секция `Services`) хранится массив ID выбранных продуктов: `[id1, id2, id5]`.
- Фронтенд делает запрос к базе, фильтрует только эти ID и отрисовывает их в заданном порядке.

## 3. Редактор Форм
- Форма — это JSON-конфиг, описывающий поля:
  ```json
  { "id": "main_form", "fields": ["name", "phone"], "hasPrivacy": true }
  ```
- Каждая кнопка на сайте имеет атрибут `formId`. При клике открывается Popup с конфигом этой формы.
